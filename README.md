# Anna Morozova Photo Landing

Статический лендинг фотографа, собранный как демонстрационный кейс для Puskweb.

## Что внутри

- адаптивный лендинг без сборщика;
- `index.html`, `styles.css`, `script.js`;
- локальные изображения в `assets`;
- мобильное меню, слайдер галереи, отзывы, раскрывающиеся FAQ и hover-эффекты;
- готово для публикации на GitHub Pages.

## Как открыть локально

Откройте `index.html` в браузере. Сборка не нужна.

## Рекомендуемое имя репозитория

`anna-morozova-photo`

## GitHub Pages

1. Создайте публичный репозиторий `anna-morozova-photo`.
2. Загрузите файлы проекта в корень репозитория.
3. В настройках репозитория откройте `Pages`.
4. Source: `Deploy from a branch`.
5. Branch: `main`, folder: `/root`.

После публикации сайт будет доступен по адресу:

`https://<github-username>.github.io/anna-morozova-photo/`
