# Anna Morozova Photo

Адаптивный сайт-портфолио фотографа Анны Морозовой: первый экран, направления съёмки, галерея работ, отзывы, FAQ и контакты.

Посмотреть сайт: https://v-kozintsev.github.io/photographer_landing_page/

## Что использовалось

- HTML, CSS и JavaScript
- Vite для локального запуска и сборки
- GitHub Pages для публикации

## Локальный запуск

```bash
pnpm install
pnpm dev
```

## Сборка

```bash
pnpm build
```
